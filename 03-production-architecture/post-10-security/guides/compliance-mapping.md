# Compliance Mapping Guide

Use this as a working bridge between the blog guidance and formal control frameworks.

| Control Theme | Hyper-V Control | Typical Mapping Language |
|---|---|---|
| Host hardening | Secure Boot, TPM, VBS, Credential Guard | Secure configuration, platform integrity |
| Data in transit | SMB encryption, cluster channel protection | Encryption in transit |
| Data at rest | BitLocker, controlled key handling | Encryption at rest |
| Access control | RBAC, JEA, remoting boundaries | Least privilege, privileged access |
| Monitoring | Event forwarding, Defender telemetry, alerting | Audit logging, detection, response |

Treat this file as a planning aid, not a claim of certification.
